\## 📌 Objective



The aim of this experiment is to design and analyze a BJT-based voltage amplifier.



The input AC signal is amplified in terms of voltage, current, or power and delivered to the output load. 



\---



\## ⚙️ Circuit Description



The amplifier circuit includes:



\* NPN transistor (BC546B)

\* Biasing resistors (470kΩ, 220kΩ, 1kΩ, 100Ω)

\* Coupling capacitors (10µF)

\* DC supply (VCC = 10V)



The circuit is first biased in DC, then an AC signal is applied to the input. 



\---



\## 🔬 Experimental Procedure



\* DC biasing voltages (VB, VC, VE) are measured

\* Collector current is calculated

\* AC input signal (1 kHz sine wave) is applied

\* Output signal is observed using oscilloscope

\* Signal distortion (clipping) is analyzed



As shown in the circuit diagram (page 2–3), proper biasing is required to avoid signal distortion.



\---



\## 📊 Analysis



\* Gain is calculated as:



```id="eq1"

Av = Vo / Vin

```



\* Output signal amplitude increases compared to input

\* Signal waveform remains sinusoidal when properly biased



From the measurements:



\* Example gain ≈ \*\*8.36\*\* (from report calculations)



\---



\## 📈 Observations



\* Amplification depends on correct biasing

\* Distortion occurs if input signal is too large

\* Output waveform shape is preserved when operating correctly

\* Gain varies depending on circuit parameters
